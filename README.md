# camera_qhyccd


说明（Readme):<br>
`camera_qhyccd`是一款可以配备[QHY 5-II系列相机](http://www.qhyccd.com/cn/QHY5-II.html)的Android应用程序，它将QHY 5-II导星相机通过OTG连接到Android设备，并使用`camera_qhyccd`应用程序来拍摄图片，极大的方便了喜欢户外探索的用户，减少他们携带电脑的不便。\<br>
